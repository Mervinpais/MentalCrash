Commands

  • '.' command: Exits the program.
  • 'p' command: Prints to the console. Usage: p "<string>"
  • 'i' command: Inputs data. Usage: i
  • 'V' command: Creates a variable based on type and data given.

Variable Creation

The 'V' command is used to create variables. There are two types of variables:

  • String (str) variables: Usage: V str e|"<string>"
  • Integer (int) variables: Usage: V int e|<number>

Example:

V str e|"Hello, World!" (Creates a string variable with the value "Hello, World!")
V int e|42 (Creates an integer variable with the value 42)
