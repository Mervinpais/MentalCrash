Commands

Function Creation

The 'f' command is used to create functions. Functions can be defined with or without parameters.

Function without parameters:

f helloWorldFunction (p "Hello World!") (Creates a function called 'helloWorldFunction' that prints "Hello World!")

Function with parameters:

f sayFunction [str e] (p e) (Creates a function called 'sayFunction' that takes a string parameter 'e' and prints its value)
f sayFunction [str e] (p "Hello World! using say function") (Creates a function called 'sayFunction' that takes a string parameter 'e' and prints a custom message)

If Statement

The 'I' command is used for conditional statements. It evaluates a condition and executes different code depending on whether the condition is true or false.

Example:

I [1 == 1, p "1 equals 1", "1 does NOT equal 1"] (Checks if 1 equals 1, and prints "1 equals 1" if true, or "1 does NOT equal 1" if false)
I [-2 == 2, p "-2 equals 2", "-2 doesn't equal 2"] (Checks if -2 equals 2, and prints "-2 equals 2" if true, or "-2 doesn't equal 2" if false)
I [1 == "1", p "1 equals \"1\"", "1 does NOT equal \"1\""]
I [true == false, p "true equals false", "true doesn't equal false"]
